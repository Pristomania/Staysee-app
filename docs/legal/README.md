# Юридические документы StaySee

## Как подключить вашу оферту (из GPTs)

1. Положите исходный текст от юриста в **`offer-source-gpts.md`** (или `.docx` → экспорт в markdown).
2. Пройдите чеклист **`OFFER_ADAPTATION.md`** — замены названий, сервиса, оплаты, лимитов.
3. Разбейте итог на секции в **`src/content/legal/offer.ts`** (массив `OFFER_SECTIONS`) — так текст показывается в приложении.
4. Укажите в `OFFER_META`: дату вступления, версию, реквизиты исполнителя (как в оферте).
5. На регистрации пользователь принимает оферту галочкой; в **Контекст** → «Публичная оферта».

Политику конфиденциальности можно вести отдельно: `privacy-policy.md` → позже `src/content/legal/privacy.ts` (сейчас — экран `PrivacyScreen` с кратким текстом).

## Файлы

| Файл | Назначение |
|------|------------|
| `offer-source-gpts.md` | **Ваш** исходник (не коммитить, если там персональные черновики — добавьте в `.gitignore` при необходимости) |
| `offer.md` | Рабочая версия после адаптации (для правки юристом) |
| `OFFER_ADAPTATION.md` | Что менять при переносе GPTs → StaySee App |

## Важно

Текст в `offer.ts` — **юридический документ**. Не генерируйте его ИИ с нуля; используйте адаптированную оферту юриста.
