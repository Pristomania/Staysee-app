## Инструкция для разработчика

### На экране регистрации — ДВА отдельных чекбокса:

**Чекбокс 1 (Пользовательское соглашение):**
```
☐ Я принимаю [Пользовательское соглашение]
```

**Чекбокс 2 (Согласие на ОПД — обязательный):**
```
☐ Я прочитал(а) и понимаю, что мои сообщения могут содержать 
  сведения о моём психологическом состоянии (специальная категория 
  по 152-ФЗ). Даю явное согласие на обработку в соответствии с 
  [Политикой обработки персональных данных]
```

### Что фиксировать в базе данных:

```sql
CREATE TABLE user_consents (
  user_id        UUID REFERENCES auth.users(id),
  consent_type   TEXT,          -- 'terms_v1', 'pd_processing_v1'
  document_hash  TEXT,          -- SHA-256 версии документа
  consented_at   TIMESTAMPTZ,
  ip_address     TEXT,
  user_agent     TEXT,
  PRIMARY KEY (user_id, consent_type)
);
```

Поле `document_hash` — доказательство в суде, что пользователь принял именно эту версию документа, а не другую.
