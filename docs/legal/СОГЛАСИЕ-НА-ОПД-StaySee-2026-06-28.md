## Инструкция по внедрению для разработчика

### На экране регистрации должно быть ДВА отдельных чекбокса:

**Чекбокс 1 (Пользовательское соглашение):**
```
☐ Я принимаю [Пользовательское соглашение]
```

**Чекбокс 2 (Согласие на ОПД — обязательный):**
```
☐ Я даю согласие на обработку персональных данных, в том числе 
  сведений о моём психологическом состоянии, в соответствии с 
  [Политикой обработки персональных данных]
```

### Что фиксировать в базе данных:

В таблице пользователей (или отдельной таблице `user_consents`) хранить:

```sql
CREATE TABLE user_consents (
  user_id        UUID REFERENCES auth.users(id),
  consent_type   TEXT,          -- 'terms_v1', 'privacy_v1', 'pd_processing_v1'
  consented_at   TIMESTAMPTZ,
  ip_address     TEXT,
  user_agent     TEXT,
  PRIMARY KEY (user_id, consent_type)
);
```

Это необходимо для подтверждения факта согласия при проверке Роскомнадзора.
